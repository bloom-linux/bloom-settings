# bloom-settings

Settings panel and system control popups for Bloom Linux.

Includes:
- **bloom-settings** — Full settings panel (display, input, users, connectivity, startup)
- **bloom-volume** — Volume mixer popup (Super+A)
- **bloom-brightness** — Brightness control popup (Super+B)
- **bloom-network** — WiFi/network manager popup (Super+W)
- **bloom-power** — Power menu
- **bloom-calendar** — Clock and calendar popup

## Install

```
pacman -S bloom-settings
```

## Dependencies

`gtk4` `python` `python-gobject` `networkmanager` `brightnessctl` `pipewire`
